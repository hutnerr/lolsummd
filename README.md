## Overview
A lightweight web app for League of Legends players to aggregate champion mastery points across multiple accounts. Currently WIP.

## Setup
If you wanted to set it up locally, you would have to provide your own API key in `data/key.txt`. You would also have to `pip install -r requirements.txt`
